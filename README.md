# Bank 애플리케이션

### 프로젝트 팁

- 테이블설계는 처음부터 완벽할 수 없다.
- 처음에는 가장 쉬운 테이블 구조 만들기
- 진행하면서 수정 보완하는 것!!

### 진행순서
- 도메인 생성
- ExceptionHandling 예전꺼에 쓸거면
- 시큐리티 설정
- logger 세팅 얘 
- DTO Validation AOP 얘 3개 세팅해야함
- 회원가입
- 로그인
- JWT 로그인으로 변경
- 서비스 기능 만들기

### 서비스 기능정리
- 회원가입
- 로그인
- 계좌등록
- 본인계좌 목록보기 (Account만 보이면 됨)
- 본인계좌 상세보기
- 본인계좌 삭제하기 (update)
- 입금하기
- 출금하기
- 이체하기
- 입출금목록보기
- 입금목록보기
- 출금목록보기
